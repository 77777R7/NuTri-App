## 🐛 **Troubleshooting**

**Port already in use?**
```bash
lsof -ti:8081 | xargs kill -9
npx expo start
```

**Clear cache:**
```bash
npx expo start --clear
```

**Reinstall:**
```bash
rm -rf node_modules
npm install
```
